# **App Name**: Saeed Store ERP Lite

## Core Features:

- Dashboard Summary: Dashboard with key summary data extracted from Excel sheets and presented in a user-friendly format.
- Table View: Display each Excel sheet (Inventory, Sales, Expenses, etc.) as an interactive HTML table with data loaded from the Excel file.
- CRUD Operations: Enable Create, Read, Update, and Delete operations on the data displayed in each table.
- Data Refresh: Allow users to upload a new Excel file to refresh the application data. Uses the new excel file after validation.
- Export to Excel: Export table data (per module) to Excel.
- Authentication: Basic login authentication for admin access.
- Logistics Optimization: A generative AI powered 'Insights Tool' to suggest the best logistics routes considering time of the year and traffic for efficiency, based on historic data from your logistics module and using external data where appropriate. This is a tool to help decide the ideal logistics solution.

## Style Guidelines:

- Primary color: Muted blue (#5DADE2) to provide a professional and calming interface, reflecting reliability and trust.
- Background color: Light gray (#F5F7FA), almost white, providing a clean and modern backdrop that ensures readability and reduces eye strain.
- Accent color: Soft green (#A3E4D7) to highlight key actions and elements, symbolizing growth and positive performance.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable interface. The font provides both clarity and a touch of warmth suitable for both headlines and body text.
- Simple, consistent icons from a library like FontAwesome or Material Icons.
- Clean and minimal layout using Bootstrap or Tailwind CSS.
- Subtle transitions for page loads and data updates.
